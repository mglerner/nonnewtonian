Embed on your own site

You can show this collection inside a page you already run. It updates as you approve submissions, and the iframe grows to fit its contents, so you don't get an inner scrollbar.

How to add it to WordPress
  1. Edit the page or post where you want it.
  2. Add a block and choose Custom HTML (search "custom html" in the block picker).
  3. Paste the snippet below into that block.
  4. Preview or publish. No plugin required.

On WordPress.com's free plan, Custom HTML blocks are limited; a self-hosted WordPress.org site or most other CMSes accept it as-is. If your host strips <script>, the collection still shows; it just won't auto-resize, so set a fixed height on the iframe.
